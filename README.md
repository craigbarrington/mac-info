# mac-info
A lightweight script to display hardware information on macOS.

## Instructions
<br>`curl -sSL https://www.craigbarrington.com/macInfo.sh | bash`<br><br>

Run the above curl command in the terminal to get the following mac information:

* Model Name
* Model ID
* Model Number
* Serial Number
* CPU
* Memory
* Disk Size
* macOS Version
* IP address
