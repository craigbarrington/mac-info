# mac-info
A lightweight script to display hardware information on macOS.

## Instructions
either download the script to your mac and run **macInfo.sh** or run the following curl command in the terminal `curl -sSL https://www.craigbarrington.com/macInfo.sh | bash` to get the following information:

* Model Name
* Model ID
* Model Number
* Serial Number
* CPU
* Memory
* Disk Size
* macOS Version
* IP address
